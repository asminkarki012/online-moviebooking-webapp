### Features
- Authentication and Authorization using JWT and RoleGuard
- OTP verification 
- Generate pdf with qr code 
- Stripe payment portal is integrated
- Can Send Ticket via Email

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
